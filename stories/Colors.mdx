# Colors

Vite treeshakable library uses [Tailwind](https://tailwindcss.com) as its CSS framework.

We are extending the default Tailwind config with the colors in the list below.  
You can use them in all Tailwind color-related classes e.g `bg-midnight`, `text-ocean` etc.
